# Succinct 調査用のディレクトリ

## インストール

```bash
curl -L https://sp1up.succinct.xyz | bash
```

以下のコマンドを実行する。

```bash
source /Users/harukikondo/.zshenv
```

以下で初期化

```bash
sp1up
```

以下で初期化

```bash
cargo prove --version
RUSTUP_TOOLCHAIN=succinct cargo --version
```

## テンプレート作成

```bash
cargo prove new --evm fibonacci
```
