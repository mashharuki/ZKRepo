## How to run

- install

  ```bash
  yarn
  ```

  ```bash
  git submodule update --init --recursive
  ```

- smartcontract build

  ```bash
  yarn backend:build
  ```

- frontend build

  ```bash
  yarn frontend:build
  ```

- frontend dev

  ```bash
  yarn frontend:dev
  ```
